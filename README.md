# luas-lingkaran

# alur algoritma


1. menetapkan nilai Phi yaitu 3.14
2. menghitung jari-jari lingkaran.
3. menghitung luas lingkaran dengan rumus L=phi*r*r
4. mencetak nilai L
5. selesai
